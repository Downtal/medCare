1. Mục tiêu hệ thống
•	Bán: 
o	thuốc không kê đơn (OTC) 
o	thực phẩm chức năng 
o	 thiết bị y tế 
•	Hỗ trợ: 
o	chatbot tư vấn sản phẩm

•	Tuân thủ: 
o	hạn chế bán thuốc kê đơn (có kiểm soát) 

2. Actors (Tác nhân)
Customer (Người dùng)
Admin
Pharmacist (Dược sĩ – optional nâng cao)

3. FLOW HỆ THỐNG
Customer Flow
1.	Truy cập website 
2.	Đăng ký / đăng nhập 
3.	Tìm kiếm sản phẩm 
4.	Chat với AI: 
o	“Tôi bị đau đầu nên dùng gì?” 
5.	AI: 
o	gợi ý sản phẩm phù hợp 
o	cảnh báo (nếu cần) 
6.	Xem chi tiết sản phẩm 
7.	Thêm vào giỏ hàng 
8.	Thanh toán 
9.	Theo dõi đơn hàng 

Flow Chatbot AI 
1.	User nhập triệu chứng / nhu cầu 
2.	NLP xử lý 
3.	Mapping: 
o	triệu chứng → nhóm sản phẩm 
4.	Trả về: 
o	danh sách thuốc phù hợp 
o	hướng dẫn sử dụng cơ bản 
o	cảnh báo 

Admin Flow
1.	Login admin 
2.	Quản lý: 
o	sản phẩm 
o	danh mục 
3.	Quản lý đơn hàng 
4.	Quản lý user 
5.	Quản lý dữ liệu chatbot 
6.	Xem báo cáo 

Pharmacist Flow (optional)
•	Duyệt đơn thuốc đặc biệt
