# Plan 10-01: Setup & Login E2E with Playwright

Mục tiêu: Thiết lập môi trường kiểm thử E2E bằng Playwright và viết kịch bản test cho luồng đăng nhập của người dùng.

---

## Các nhiệm vụ

### Task 1: Cài đặt và Cấu hình Playwright
- Cài đặt Playwright: `npx playwright install` (và các dependencies).
- Cấu hình `playwright.config.ts`: base URL (`http://localhost:3000`), timeout, browsers (Chromium).
- Tạo cấu trúc thư mục `e2e` trong `FE`.

### Task 2: Viết kịch bản Login E2E
- Truy cập trang chủ -> Click Đăng nhập.
- Nhập thông tin tài khoản hợp lệ.
- Xác nhận chuyển hướng về trang chủ và hiển thị tên người dùng/avatar.
- Kiểm tra trường hợp đăng nhập thất bại với mật khẩu sai.

### Task 3: Chạy test và báo cáo
- Chạy lệnh `npx playwright test`.
- Kiểm tra HTML report được tạo ra.

---

## Tiêu chí hoàn thành

- [ ] Playwright được cài đặt và cấu hình đúng.
- [ ] Kịch bản `login.spec.ts` pass 100% trên trình duyệt Chromium.
- [ ] HTML report hiển thị kết quả kiểm thử rõ ràng.

```bash
cd FE
npx playwright test
```
