# Plan 09-01: Setup & Basic UI Tests for Frontend

Mục tiêu: Thiết lập môi trường kiểm thử cho React (Vite) bằng Vitest và React Testing Library, đồng thời viết các bài test cơ bản cho các component UI.

---

## Các nhiệm vụ

### Task 1: Cài đặt và Cấu hình Vitest
- Cài đặt dependencies: `vitest`, `@testing-library/react`, `@testing-library/jest-dom`, `jsdom`.
- Cấu hình `vite.config.ts` để hỗ trợ testing.
- Tạo file `src/test/setup.ts` để cấu hình môi trường global cho tests.

### Task 2: Viết Unit Tests cho UI Components
- **Button Component**: Kiểm tra sự kiện click và hiển thị label.
- **Input Component**: Kiểm tra việc nhập dữ liệu và hiển thị lỗi validation.
- **Header/Footer**: Đảm bảo hiển thị đúng thông tin điều hướng.

### Task 3: Chạy test và xác nhận
- Cập nhật `package.json` với lệnh `npm test`.
- Đảm bảo tất cả tests cơ bản đều pass.

---

## Tiêu chí hoàn thành

- [ ] Lệnh `npm test` chạy thành công.
- [ ] Ít nhất 3 UI components có unit tests đạt tỉ lệ thành công 100%.
- [ ] Cấu hình setup.ts hoạt động đúng (không lỗi mock global).

## Lệnh kiểm tra
```bash
cd FE
npm test
```
