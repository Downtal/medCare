# Plan 09-02: Complex Component Tests for Frontend

Mục tiêu: Viết các bài test cho các component có logic phức tạp hơn như CartDrawer và ProductCard, sử dụng Mocking cho Store (Zustand) và Session (Next-Auth).

---

## Các nhiệm vụ

### Task 1: Viết Test cho CartDrawer
- Mock `useCartStore` để giả lập dữ liệu giỏ hàng.
- Kiểm tra việc hiển thị danh sách sản phẩm.
- Kiểm tra tính năng cập nhật số lượng và xóa sản phẩm khỏi giỏ hàng.

### Task 2: Viết Test cho ProductCard
- Kiểm tra hiển thị thông tin sản phẩm (tên, giá, ảnh).
- Kiểm tra tính năng "Thêm vào giỏ hàng" và gọi action tương ứng.
- Kiểm tra hiển thị nhãn "Hết hàng" hoặc "Thuốc kê đơn".

### Task 3: Chạy toàn bộ bộ test
- Chạy `npm test` để đảm bảo 100% test cases đều pass.
