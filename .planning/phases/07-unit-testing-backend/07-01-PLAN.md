### Task 3: Viết OrderServiceTest.java

**File:** `BE/order-service/src/test/java/com/medcare/orderservice/service/OrderServiceTest.java`

```java
@ExtendWith(MockitoExtension.class)
class OrderServiceTest {

    @Mock private OrderRepository orderRepository;
    @Mock private CartService cartService;
    @Mock private ProductClient productClient;
    @Mock private PromotionClient promotionClient;
    @Mock private ShippingClient shippingClient;
    @Mock private InventoryClient inventoryClient;
    @Mock private OrderStrategyFactory strategyFactory;

    @InjectMocks private OrderService orderService;

    // Test cases:
    // 1. shouldCreateOrderSuccessfullyWhenItemsProvidedInRequest
    // 2. shouldFallbackToCartWhenNoItemsInRequest
    // 3. shouldThrowExceptionWhenCartIsEmpty
    // 4. shouldCalculateTotalAmountCorrectly
    // 5. shouldApplyDiscountWhenVoucherCodeProvided
    // 6. shouldGenerateUniqueOrderCode
    // 7. shouldSaveOrderWithCorrectStatus
}
```

**Chi tiết test cases:**

**Test 1 — Happy path (items từ request):**
```java
@Test
void shouldCreateOrderSuccessfullyWhenItemsProvidedInRequest() {
    // Given
    OrderRequest request = buildOrderRequest(List.of(new OrderItemRequest(1L, 2, "hộp")));
    ProductClient.ProductDto product = buildProductDto(1L, "Paracetamol", BigDecimal.valueOf(50000));
    when(productClient.getProductById(1L)).thenReturn(product);
    when(orderRepository.save(any())).thenAnswer(inv -> inv.getArgument(0));
    
    // When
    Order result = orderService.createOrder("user1", request);
    
    // Then
    assertNotNull(result);
    assertEquals(BigDecimal.valueOf(100000), result.getTotalAmount());
    verify(orderRepository).save(any(Order.class));
}
```

**Test 2 — Fallback to cart:**
```java
@Test
void shouldFallbackToCartWhenNoItemsInRequest() {
    // Given - request không có items
    OrderRequest request = buildOrderRequest(null);
    CartDto cart = buildCartWithItems();
    when(cartService.getCart("user:user1")).thenReturn(cart);
    when(productClient.getProductById(any())).thenReturn(buildProductDto(...));
    when(orderRepository.save(any())).thenAnswer(inv -> inv.getArgument(0));
    
    // When
    Order result = orderService.createOrder("user1", request);
    
    // Then
    assertNotNull(result);
    verify(cartService).getCart("user:user1");
}
```

**Test 3 — Empty cart exception:**
```java
@Test
void shouldThrowExceptionWhenCartIsEmpty() {
    // Given
    OrderRequest request = buildOrderRequest(null);
    CartDto emptyCart = new CartDto();
    emptyCart.setItems(List.of());
    when(cartService.getCart(any())).thenReturn(emptyCart);
    
    // When/Then
    assertThrows(RuntimeException.class, () -> orderService.createOrder("user1", request));
}
```
