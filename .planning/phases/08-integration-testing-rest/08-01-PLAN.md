# Plan 08-01: Integration Tests for Order Service

Mục tiêu: Kiểm thử tích hợp các endpoint của `order-service`, bao gồm luồng đặt hàng và quản lý đơn hàng.

---

## Các nhiệm vụ

### Task 1: Chuẩn bị dữ liệu mẫu (test-data.sql)
Tạo file `BE/order-service/src/test/resources/data-test.sql` để khởi tạo dữ liệu đơn hàng và cấu hình test.

### Task 2: Viết OrderControllerIntegrationTest.java
Thực hiện gọi API thực tế tới `/api/orders` bằng MockMvc.

**Test cases:**
1. **POST /api/orders/checkout:**
   - Case: Tạo đơn hàng thành công với JWT token hợp lệ.
   - Case: Trả về 401 khi không có token.
   - Case: Trả về 400 khi payload không hợp lệ.
2. **GET /api/orders/me:**
   - Case: Lấy danh sách đơn hàng của người dùng hiện tại.
3. **GET /api/orders/{id}:**
   - Case: Lấy chi tiết đơn hàng thành công.
   - Case: Trả về 404 khi đơn hàng không tồn tại.

### Task 3: Viết CartControllerIntegrationTest.java
Kiểm tra tích hợp với Redis (giả lập hoặc dùng in-memory redis nếu cần, nhưng do đã có Unit Test Redis mạnh mẽ, Integration này sẽ tập trung vào sự kết nối Controller -> Service).

---

## Tiêu chí hoàn thành

- [x] File `data-test.sql` được tạo.
- [x] `OrderControllerIntegrationTest.java` hoàn thành với ít nhất 4 test cases.

```bash
gradlew order-service:test --tests "com.medcare.orderservice.controller.*"
```
